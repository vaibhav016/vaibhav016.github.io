  Two classification projects: (1) Emotion classification from speech using the emo-db dataset,
  and (2) Guitar chords classification using a custom synthesized dataset. Both use MFCC feature
  extraction fed into deep convolutional networks to learn spectral and temporal audio features.
tech:
  - Deep Learning

Both projects work on similar lines. First, feature extraction is made from speech using MFCC. Then those features are fed to a deep convolution classification network which can learn the spectral as well as the temporal features from audio.

The emotion classification used emo-db dataset, which is highly curated and made for academic research. For guitar chord classification, I had to synthesize a large dataset for guitar chords. Since they were undergraduate projects, these projects made the foundation for machine learning and deep learning.
